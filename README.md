# Multivariate Statistical Analysis with Machine Learning

This project explores multivariate statistical techniques using R.  
The analysis focuses on identifying patterns in high-dimensional data and applying statistical learning methods.

## Techniques Used

The project applies several multivariate methods, including:

- Data preprocessing and cleaning
- Exploratory data analysis
- Dimensionality reduction
- Clustering methods
- Statistical interpretation of multivariate data

## Tools

- R
- R Markdown
- Statistical learning methods
- Data visualisation

## Files

digit_unsupervised_learning.Rmd  
Contains the full analysis workflow and code.

digit_unsupervised_learning.pdf  
Compiled report of the analysis and results.

digit.txt  
Dataset used in the analysis.
